# 🐦 Flappy Bird Clone (Python + Tkinter)

A simple **Flappy Bird Clone** built using **Python** and **Tkinter**.

This project recreates the classic Flappy Bird gameplay with smooth animations, randomly generated pipes, collision detection, score tracking, and restart functionality—all without using any external game libraries.

## 📖 How the Game Works

- Gravity constantly pulls the bird downward.
- Press **Space** or **Up Arrow** to make the bird flap upward.
- Pipes are generated randomly and move from right to left.
- Passing through a pipe increases your score.
- Hitting a pipe, the ground, or the top of the screen ends the game.
- Press **R** or **Space** after Game Over to start a new game.
